# 实现 (5).add(3).minus(2) 功能。

更详细的解法：实现 (5).add(3).minus(2) 功能 (opens new window)

Number.prototype.add = function(number) {
  return this + number
}
Number.prototype.minus = function(number) {
  return this - number
}
